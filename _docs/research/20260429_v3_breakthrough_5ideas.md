# 한국 공공데이터 다중결합 획기적 비즈니스 아이디어 5건 (v3)

> 작성일: 2026-04-29
> 원 프롬프트: `_docs/prompts/20260429_first_prompt.md` + `_docs/prompts/20260429_prompt.md`
> v2와의 차이: v2는 "v1 데이터셋 표 위에서 결합 강도를 높였다"면, v3는 **v2가 다루지 않은 새로운 부처·기관 데이터** (한국전력·법원·조달청·국립환경과학원·산업안전보건공단·KIPRIS·해양수산부·NTIS 등)를 적극 발굴해 **"이걸 누가 안 만들었지?" 수준의 의외성·사회 트렌드 정확 매핑·시장 임팩트**를 추가했다.
> 선정 기준: ① **결합 의외성** (단일 데이터로 안 보이고, 결합 발상 자체가 산업적으로 미사용) ② **사회 트렌드 정확 매핑** (중대재해법·ESG·고령화·기후위기·자영업 위기 등 지금 이 시점) ③ **1인 개발자 6개월 MVP 가능성** ④ **결제 의향 명확한 시장**

---

## TL;DR (3줄)
- v3은 **에너지·법원·환경의학·공공조달·산업안전** 다섯 가지 **v2 미사용 도메인**의 데이터를 결합한 5건이다.
- 가장 획기적인 단일 베팅은 **③ PowerEcon (한국전력 시간대별 전력 → 골목 경제활동 실시간 지표)** — 미국 SafeGraph·Placer.ai에 해당하는 **한국형 alternative data 시장의 명백한 공백**을 채우는 발상.
- 가장 큰 시장은 **④ BidIntel (200조 공공조달 입찰 인텔리전스)**, 가장 핫한 트렌드는 **⑤ SafetyScore (중대재해법·ESG)**.

## 아이디어 ①. **"우리 동네 환경건강 위험 지도" — EnviroHealth**
### 결합 메커니즘: 환경 데이터 × 의료 통계 × 인구

1. **One-liner**: **부동산 매수·전세 검토자, 자녀 보호 학부모, 보험사**에게 **거주지 반경 5km 화학물질 배출·미세먼지·진료통계·검진 데이터를 결합한 5년 건강 위험 점수와 핵심 위험 요인 시각화**를 제공한다.

2. **사용 데이터셋**:
   - **국립환경과학원 PRTR 화학물질 배출량** (`ncis.nier.go.kr`) — 사업장 단위, 1급 발암물질 포함
   - 환경부 대기오염 (#6 v1 표) — PM2.5·NO2·O3 등
   - **국립환경과학원 토양·지하수 오염도**
   - **HIRA 진료통계** — 호흡기·심혈관·암 진료율, 시군구·연령
   - **건강보험공단 검진통계 NHIS** — 시군구·연령
   - KOSIS 인구·연령·소득 (#10)
   - 도로명주소 (#3)
   - 결합: **국가화재정보시스템** (소방청) — 화학공장 사고이력
   - 결합: 기상청 (#1) — 풍향·풍속 (오염물질 확산 추정)

3. **🔑 결합 의외성 메커니즘 (왜 단일 데이터로 불가능한가)**:
   - PRTR 단독 → "이 사업장이 1급 발암물질 X kg 배출"만 앎 (위험 영향권 모름)
   - 진료통계 단독 → "이 시군구 폐암 발병률 +18%"만 앎 (원인 미상, 책임 불가)
   - 결합 → **"당신 거주지 반경 5km PRTR 1급 사업장 3개 + 5년 평균 PM2.5 28㎍/㎥ + 풍향(편서풍) 노출 220일/년 + 시군구 호흡기 진료율 +18% (전국 대비) → 5년 건강위험 점수 73/100, 주요 기여 요인: ① 화학사업장 A (편서풍 직격) ② 도로 NO2 ③ 노후 주거"** 형태의 **귀속·해석 가능한 점수**
   - 글로벌 벤치마크: 미국 ProPublica의 "Toxmap"·EPA의 EJSCREEN — **한국에는 미존재**
   - 결합이 빠지면 가치의 **약 90% 손실** (단일 데이터는 모두 무료 공개지만 위험 귀속 불가)

4. **타겟 사용자 & 페르소나**:
   - 1차: **자녀(0~10세) 양육 30~40대 부모** — 매수·전세 결정 시 학군·평수 외 **환경 건강** 평가가 신뢰할 자료가 없음
   - 2차: **민영 보험사** — 가입 심사·갱신 요율 보조지표 (실손·암·종신)
   - 3차: **환경 NGO·기자** — 무료 + viral 채널
   - 4차: **지자체 보건소·환경정책팀** — B2G

5. **수익모델 (구체 수치)**:
   - B2C 1회 리포트: **9,900원** (좌표 1개) / Pro **월 4,900원** (관심 지역 변동 알림 + 주변 매물 점수)
   - 보험사 API: 호출당 **100원** 또는 월 정액 999만원
   - PropTech (직방·다방·디스코) API: 좌석 또는 호출당
   - 지자체 B2G: 연 600만원 (정책 자료)
   - 가정: 1년차 B2C 8,000건/연 + Pro 3,000명 + 보험사 2개 + PropTech 1개 = MRR 약 2,800만원

6. **차별화 포인트 (moat)**:
   - 글로벌은 EJSCREEN(EPA)·ProPublica가 있으나 **한국 특화 + 한국어 + 결제 가능 SaaS는 부재**
   - **데이터 매월 갱신 → 동적 점수**가 자산 (정적 리포트 대비)
   - 환경 NGO·시민 검증 채널이 자체 viral 마케팅 (CAC 거의 zero)
   - 시간 갈수록 **건강 영향 추적 라벨** (예측 점수 vs 실제 진료율 변화) 자산화

7. **기술 난이도 & MVP 개발 기간**: **6개월 (HIGH)**
   - 1단계 (2개월): 5개 데이터 ETL + 시군구·좌표 정규화 + 풍향 확산 모델
   - 2단계 (2개월): 위험 점수 가중치 모델 (학계·환경의학 자문 1~2회)
   - 3단계 (1.5개월): Next.js 지도 dashboard + 리포트 PDF
   - 4단계 (0.5개월): 결제·알림

8. **리스크**:
   - 진료통계의 시군구 단위 가용성 — HIRA 정형 데이터로 충분하나 읍면동 단위는 협력 필요
   - **부동산 가격 하락 영향 클레임** 위험 → 환경 NGO 후원·중립 데이터 표시 명분 필수
   - 화학물질 배출 사업장의 정밀 좌표·풍향 모델의 가정 명시 필요
   - 보험사 활용이 가입 차별로 비춰질 수 있음 — 약관 검토

## 아이디어 ②. **"법원 경매 매물 함정 자동 분석" — AuctionLens**
### 결합 메커니즘: 경매 × 등기 × 부동산 × 다층 위험

1. **One-liner**: **개인 경매 입찰자, 경매 컨설팅·법무사, 부동산 PropTech**에게 **대법원 경매·등기부·실거래·인근 폐업·도시계획·건축물 안전·재해위험 데이터를 결합한 매물별 위험·잠재가치 점수와 자동 권리분석**을 제공한다.

2. **사용 데이터셋**:
   - **대법원 경매정보** (`courtauction.go.kr`) — 매각물건·권리신고·기일·결과
   - **대법원 인터넷등기소** (`iros.go.kr`) — 등기부 (가등기·근저당·법정지상권)
   - 국토부 부동산 실거래 (#2 v1) — 인근 매매·전세
   - 사업자 등록·폐업 (#4 v1) — 점유자/상권 분석
   - 도로명주소 (#3 v1)
   - **국토부 토지이용계획** (`luris.molit.go.kr`) — 용도·도시계획 변경
   - **국가공간정보 건축물대장** — 안전등급·경과년수·구조
   - 국민재난안전포털 — 재해위험구역·침수
   - **소방청 화재발생** — 사고 이력
   - KOSIS 인구이동·연령

3. **🔑 결합 의외성 메커니즘**:
   - 지지옥션·옥션원 등 기존 사이트 → **정보 제공만**, 권리분석은 변호사·컨설턴트에 의존
   - 단일 등기부 → "근저당 5건"만 앎, 위험 분석 X
   - 결합 → **"이 매물의 진짜 가치 = 인근 시세 92%, 권리분석 함정 3개 (가등기 의심·법정지상권 가능성·점유자 사업자 폐업 임박), 5년 후 도시계획 변경 가능성 中, 안전등급 D, 재해위험구역 인접 → 종합 점수 41/100, 입찰가 권장 감정가의 78% 이하"**
   - LLM이 **등기부 OCR + 권리분석 + 판례 매칭 + 다층 위험**을 자동화 → 변호사 자문 1건 50~200만원 → SaaS로 1/100 비용
   - 결합이 빠지면 가치의 **약 80% 손실**

4. **타겟 사용자 & 페르소나**:
   - 1차: **개인 경매 입찰자** (연 약 8만건 매물 → 잠재 사용자 수십만, 지지옥션 유료회원만 약 5만 추정) — 권리분석 1건 50만원 부담
   - 2차: **경매 컨설팅·법무사** (전국 약 8,000명) — 고객 매물 일괄 분석 도구
   - 3차: **부동산 PropTech** (직방·디스코·호갱노노) — 경매 카테고리 강화 API

5. **수익모델 (구체 수치)**:
   - B2C 1회 리포트: **9,900원** (1매물) / 패키지 5건 39,000원
   - Pro 월 19,900원 (관심 매물 변동 알림 + 무제한 분석)
   - B2B 컨설턴트·법무사 좌석 월 99,000원 (월 100건)
   - PropTech API 호출당 200원
   - 가정: 1년차 B2C 12,000건/연 + Pro 2,000명 + 컨설턴트 200명 + API 1개 = MRR 약 4,000만원

6. **차별화 포인트 (moat)**:
   - 지지옥션·옥션원 = 정보 제공 (구독료 월 20~50만원, 그러나 분석 X)

   - 3차: **민영 보험사** — 산재보험·기업종합보험 요율
   - 4차: **정부 사업 평가** (B2G) — 공공 발주 협력사 평가

5. **수익모델 (구체 수치)**:
   - 대기업 B2B: **연 1,200~6,000만원** (협력사 100~5,000개 모니터링 단계별)
   - 컨설턴트 좌석: 월 299,000원
   - 보험사 API: 호출당 200원 또는 월 정액 999만원
   - B2G: 연 1,000~3,000만원
   - 가정: 1년차 대기업 30개 (평균 250만원/월) + 컨설턴트 50개 (30만원/월) + 보험사 1개 = MRR 약 1억 100만원

6. **차별화 포인트 (moat)**:
   - **중대재해법 시행 후 모든 발주사가 협력사 안전 평가 페인** → 시장 명확
   - 글로벌 ESG 평가사가 중소 사각지대를 안 다룸 → 한국 특화 = 진입장벽
   - 시간 갈수록 **사고 발생 검증 라벨** 자산화
   - 정부 정책 (ESG 공시 의무화 2025+) 트렌드 정확 매핑

7. **기술 난이도 & MVP 개발 기간**: **6개월 (HIGH)**
   - 1단계 (2개월): 7~8개 데이터 ETL + 회사 매칭 (사업자번호 키)
   - 2단계 (2개월): 산재 추정 모델 + ESG 점수 (도메인 자문 1~2명 필수)
   - 3단계 (1.5개월): 대시보드 + API
   - 4단계 (0.5개월): 보고서 생성·결제·영업
   - **공동창업자 권장**: 산업안전·ESG 도메인 전문가 1인

8. **리스크**:
   - 산재 통계의 **회사 단위 가용성** — 직접 데이터는 업종·지역만, 회사 단위는 추정 모델로 보완
   - **ESG 점수의 평가 방법론 신뢰성** → 학계·전문가 자문 + 방법론 투명 공개 필수
   - **공정거래법 (협력업체 차별 우려)** → 평가 도구가 발주사 의사결정 보조이지 강제 아님 명시
   - 대기업 영업 사이클 길음 (6개월+) → 컨설턴트 채널 우선
   - 평가가 부정적인 협력사로부터 명예훼손 클레임 가능성 → 데이터 출처·산정 근거 항상 동시 제공

2. **한국에 alternative data 전문 회사 부재** — 외산이 한국 시장 약함 = 명백한 공백
3. **B2B API 모델 → ARPU 단계적 상승**: 자영업 99,000원 → PropTech 99~999만원 → 헷지·증권 5,000만~3억원

6. 헷지·증권 채널 도달이 어려움이 단점이나, **PropTech·자영업·B2G 채널만으로도 MRR 3,000만원** 수준 달성 가능 → 실패 시 완전 손실 X

## 핵심 통찰 — v3에서 발견한 한국 공공데이터의 4가지 사각지대

### 사각지대 ①. 한전·가스·수자원 = 한국 alternative data 시장 공백
미국에는 SafeGraph·Placer.ai 같은 alt-data 전문 회사가 평가 1조원 단위로 존재하나, 한국은 같은 발상의 사적 데이터 회사도 부재. **한전 OpenAPI는 이미 개방돼 있는데 산업적 활용은 KEPCO 자체 보고서 외 거의 없음**. → PowerEcon

### 사각지대 ②. 환경과학원 PRTR + 의료 통계 = ProPublica·EPA 한국판 부재
미국에는 EPA EJSCREEN, ProPublica Toxmap 등이 환경 정의(Environmental Justice) 도구로 존재. 한국은 PRTR + HIRA가 모두 공개돼 있지만 **결합 + 위험 귀속 점수화는 학술논문 수준에서만 존재**. → EnviroHealth

### 사각지대 ③. 중대재해처벌법(2022) + ESG 공시(2025+) = 협력업체 평가 SaaS 부재
중대재해법 시행 후 발주 대기업 모두가 협력사 안전 평가 페인을 안고 있으나, **글로벌 ESG 평가사가 중소 사각지대를 안 다루는 동안 한국 특화 솔루션 미존재**. 시장 폭발기. → SafetyScore

### 사각지대 ④. 200조 공공조달 = 매칭은 있어도 자동작성·낙찰가 예측 부재
나라장터 OpenAPI는 정형 데이터 잘 갖춰져 있고 비즈매칭 등 큐레이션 SaaS는 있으나, **LLM이 입찰서 자동작성 + 결과 학습으로 낙찰가 예측하는 인텔리전스는 한국에 부재**. → BidIntel
