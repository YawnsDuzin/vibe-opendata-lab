# 한국 공공데이터 다중결합 시너지 비즈니스 아이디어 5건 (v2)

> 작성일: 2026-04-29
> 원 프롬프트: `_docs/prompts/20260429_first_prompt.md` + `_docs/prompts/20260429_prompt.md`
> v1과의 차이: v1 5건은 본질적으로 "단일 데이터셋 + 알림" 구조였다. v2는 **3개 이상의 데이터를 결합해야만 비로소 가치가 발생하는** 아이디어만 선정했다. 단일 데이터로 80% 이상 구현 가능한 아이디어는 제외.
> 선정 기준: ① 결합 시너지 강도 (단일 데이터로 대체 불가) ② 1인 개발자 6개월 MVP 가능성 ③ 한국 공공데이터의 실제 가용성 ④ B2B 결제 의향 또는 명확한 B2C 페인.

---

## TL;DR (3줄)
- v2는 **시간 차원(선행지표) · 공간 다층 · 이종 도메인 교차 · 정책 인과 · 프로필↔공고 메타매칭** 다섯 가지 결합 패턴을 1:1로 매핑한 5건이다.
- 가장 강력한 단일 베팅은 **③ "골목 신호등" (공간 다층 점수화)** — 7개 레이어가 동일 좌표에 모이는 순간 점수가 살아나며, 시간 갈수록 모이는 데이터가 자산이 된다.
- 가장 의외성·임팩트가 큰 건 **⑤ "정책 ROI 트래커"** — 지자체 226개라는 명확한 B2B 시장 + 시민/기자/국회의 무료 viral 채널 결합 가능.

## 아이디어 ①. **"골목 폐업 6개월 선행 경보" — CloseWatch**
### 패턴 A: 선행지표 합성 (Leading Indicator)

1. **One-liner**: **상가 임대인·프랜차이즈 본사·신규 진입 자영업자**에게 **개별 점포의 6개월 후 폐업 확률을 사업자 상태·인근 신규개업·인구이동·임대료·카드매출 변동의 합성지표로 사전 경보**한다.

2. **사용 데이터셋**:
   - 국세청 사업자 상태 (Step1 표 #4) — 점포·인근 폐업/휴업/신규
   - 도로명주소 (#3) — 동일 건물·인근 200m 사업자 그리드 매핑
   - KOSIS 인구이동·연령·소득 (#10) — 행정동 단위 분기 변동
   - 국토부 상가·오피스텔 실거래·임대료 (#2 확장: `data.go.kr/15058016` 상업업무용 부동산 실거래)
   - KAMIS·외식 물가지수(KOSIS) — 외식업 한정 비용 충격 지표
   - 결합(외부): 신용카드사 지역·업종 매출지수 (한국은행·BC카드 분기 공시 — 무료, 정형)

3. **🔑 결합 시너지 메커니즘 (왜 단일 데이터로 불가능한가)**:
   - 사업자 상태 단독 → "지금 영업 중"만 알 수 있음 (사후 판단)
   - 신규개업률 단독 → "이 동네 뜨고 있다"만 알 수 있음 (방향만)
   - **세 가지를 시계열로 결합해야** "내 점포 인근 200m에서 동종업종 신규 +30%·폐업 +15%·인구 -5%·임대료 +12% → 6개월 후 본 점포 폐업 확률 38%"라는 **숫자가 나온다**.
   - 학술 근거: 서울신용보증재단·서울시·중기부 폐업 예측 모델 다수 존재 (AUC 0.7±0.05). 모델은 공개돼 있으나 **개별 점포 단위 SaaS 형태로는 미존재**.

4. **타겟 사용자 & 페르소나**:
   - 1차: **상가 임대인 / 부동산 자산운용** — 보증금 사고·공실 리스크 회피 (서울 임대 사업자 약 70만)
   - 2차: **프랜차이즈 본사 가맹점 관리팀** (CJ푸드빌·교촌 등 점포 100개+ 본사 약 800개)
   - 3차: **자영업 신규 진입자** (입지 자가검증)

5. **수익모델 (구체 수치)**:
   - B2B 임대인 SaaS: 건물·점포 등록 단위, **월 9,900원/점포** 또는 **월 49,900원/건물(점포 10개까지)**
   - 프랜차이즈 본사 패키지: 점포 100개당 **연 600만원** (월 50만원), 본사 8개 도입 = 연 4,800만원
   - 부동산 컨설팅·중개 좌석: 월 99,000원/좌석
   - 가정: 1년차 임대인 300명 + 본사 5개 = MRR 약 700만원
   - 추가: 폐업 예측 점수 API를 부동산 PropTech(직방·다방·디스코)에 B2B API 판매 — 호출당 50원

6. **차별화 포인트 (moat)**:
   - 폐업 통계는 행정안전부·서울시 단위로 공개되지만 **점포 단위 예측 SaaS는 미존재**
   - 시간 갈수록 **자체 라벨 데이터 축적**(예측 → 실제 폐업 결과 검증)으로 모델 정확도 격차 점진 확대 — 후발 진입자가 따라잡으려면 다시 1~2년 라벨링 필요
   - 한국감정원·서울신용보증재단이 비슷한 모델을 가지고 있어도 **공공기관 특성상 SaaS화·타겟별 UI는 안 함** → 1~2년 안전

7. **기술 난이도 & MVP 개발 기간**: **5개월 (MEDIUM)**
   - 데이터 수집 (4종 일배치 + KOSIS 분기 ETL): 1.5개월
   - 그리드 매핑 + 폐업 예측 모델 (LightGBM 베이스라인 → 점진 개선): 1.5개월
   - 대시보드(Next.js + 지도 라이브러리): 1.5개월
   - 결제·알림: 0.5개월

8. **리스크**:
   - 폐업 예측이 자기실현 예언이 될 수 있음 → 임대인이 사용 시 임차인 권리 침해 우려, **임차인 식별 정보는 표시하지 않고 점포(주소) 단위 점수만 노출**하는 설계 필요
   - 신용카드 매출지수가 분기 단위라 신호 지연(최대 90일) 발생 — 보완은 SK텔레콤 유동인구 등 유료 데이터로 가능하나 비용 증가
   - 공공기관(서울신용보증재단)이 동일 SaaS를 무료 출시할 가능성 — 하지만 226개 지자체에 동일 품질 보급은 5년 이상 소요 추정

## 아이디어 ②. **"약국별 이번 주 수급 부족 예측" — RxFlow**
### 패턴 B: 이종 도메인 교차 신호 (Cross-domain Signal)

1. **One-liner**: **전국 25,000개 약국 약사·의약품 도매상**에게 **기상·질병 발생·처방 통계·회수 정보를 결합해 다음 주 본인 약국에서 부족할 의약품 Top 10과 대체 추천**을 제공한다.

2. **사용 데이터셋**:
   - 기상청 단기예보 (#1) — 독감·알러지·천식 시즌 예측
   - 식약처 의약품 안전·회수 (#7)
   - HIRA 약국·요양기관 위치 (#8)
   - 결합: **질병관리청 감염병 발생 통계** (`data.go.kr/15043207` 인플루엔자 표본감시) + **HIRA 처방통계** (`opendata.hira.or.kr` 진료내역) — 시군구·연령·질병 단위
   - 결합: 지자체 학교 결석률 (서울교육청 등) — 독감 선행지표 (의외성 ★★★)

3. **🔑 결합 시너지 메커니즘**:
   - 기상 단독 → "이번 주 기온 -5도, 미세먼지 나쁨"만 앎
   - 처방통계 단독 → "지난주 천식약 +20%"만 앎 (이미 늦음)
   - **결합 시**: "기온 급강하 + 학교 결석률 +15% + 인플루엔자 표본양성률 8% → **다음 주 시군구 단위 항생제·해열제 처방 +35% 예상 → 본 약국 재고 7일분 부족** → 도매 발주 자동 추천"
   - **선행 1~2주** 신호 → 도매 발주 리드타임(평균 2~3일) 안에 대응 가능
   - 추가 회수 발생 시 "대체 가능 동일 성분 의약품" 자동 매칭 (KD코드 기반) — 회수 데이터와 결합

4. **타겟 사용자 & 페르소나**:
   - 1차: **개인약국 약사** (체인 제외, 전국 약 18,000개) — 현재 도매 발주는 경험·감으로 결정, 재고 부족·과잉 빈번
   - 2차: **지역 의약품 도매상** (전국 약 800개) — 시군구 단위 수요 예측 (B2B로 더 큰 ARPU)

5. **수익모델 (구체 수치)**:
   - 약국 SaaS: **월 49,000원** (수요 예측 + 발주 추천 + 회수 대체)
   - 도매상 B2B: **월 299,000원** (시군구 단위 dashboard + API)
   - 가정: 1년차 약국 200곳 + 도매 30곳 = (200×49,000) + (30×299,000) = MRR 약 1,880만원
   - 추가: 도매상 → 약국 발주 중개 수수료 0.5% (옵션, v2)

6. **차별화 포인트 (moat)**:
   - 약학정보원·메디게이트는 의료진 정보 제공 채널이지 **수요 예측 SaaS 아님**
   - 약국 ERP(팜웰·약솔루션 등)는 단순 재고관리 — **외부 신호 결합 안 함**
   - 학교 결석률 + 기상 + 표본감시 + 처방을 동시에 결합하는 발상이 산업적으로 미사용 (학술논문 수준에서만)
   - 약국·도매가 누적된 발주 데이터를 모델에 피드백 → **약국별 personalized 모델로 격차 확대**

7. **기술 난이도 & MVP 개발 기간**: **6개월 (MEDIUM-HIGH)**
   - 위험요소: HIRA 처방통계는 월 단위 정형 데이터지만 **시군구 단위 세분화는 추가 협력 필요**할 수 있음 — MVP는 광역시도 단위로 시작
   - 학교 결석률은 일부 교육청만 공개 — 서울/경기 우선
   - 모델: 시계열(Prophet/NeuralProphet) + 외부 회귀변수

8. **리스크**:
   - HIRA 데이터의 세분화 한계 → MVP에서 "광역시도 단위" 제공만 가능할 수 있음
   - 학교 결석률 데이터 가용성·갱신 주기 편차
   - 약국 약사의 IT 결제 의향 불확실 (조사 필요)
   - 의약품 도매업체와 약국이 결합되면 **공정거래법 검토** 필요 (수직결합)

## 아이디어 ③. **"골목 신호등" — StreetLens** ★ 단일 베팅 추천
### 패턴 C: 공간 다층 점수화 (Geo Stack)

1. **One-liner**: **신규 창업자·부동산 컨설팅·프랜차이즈 본사 입지팀**에게 **50m × 50m 그리드에 7~10개 공공데이터 레이어를 중첩한 업종별 5년 생존 점수 + 핵심 위험·기회 요인 시각화**를 제공한다.

2. **사용 데이터셋** (모든 레이어가 동일 좌표계로 정규화됨):
   - 사업자 등록·폐업 (#4) — 업종별 grid density / 회전율
   - 도로명주소 + 행안부 주소 변경 이력 (#3) — 재개발·도시계획 신호
   - KOSIS 인구·연령·1인가구·외국인 (#10)
   - 통계청 가구이동 (`kosis.kr` 인구이동 통계)
   - 기상청 미세먼지·기온 (#6) — 야외업종 가산점/감점
   - **국민재난안전포털 재해위험지구·침수예상도** (`safekorea.go.kr` Open API)
   - **소방청 화재발생·교통사고** (`data.go.kr/15012948`)
   - 어린이보호구역·학교 위치 (교육부)
   - 결합 옵션: SK텔레콤·KT 유동인구 (유료 — Pro 플랜 한정)

3. **🔑 결합 시너지 메커니즘**:
   - 부동산 가격 + 인구만 결합한 호갱노노·디스코 — **거주용 입지 평가**에 최적화
   - 카카오맵·네이버지도 — **목적지 검색**용
   - StreetLens는 **상업/창업 입지 평가**에 특화: 7~10개 레이어가 동일 좌표에 모여 "이 좌표에서 카페를 차리면 5년 생존율 41%, 핵심 위험은 ① 동종업종 회전율 78% ② 1인가구 비중 -12% (3년 추세) ③ 침수위험구역 인접" 형태의 **귓속말급 인사이트** 제공
   - 단일 레이어로는 모두 무료 공개 데이터지만 **그리드 매칭 + 정규화 + 모델링은 미구축** — 기술적·데이터 가공 진입장벽

4. **타겟 사용자 & 페르소나**:
   - 1차: **신규 창업자** (자영업 연 80만 신규 창업 중 외식·소매 30만) — 1회 리포트 결제
   - 2차: **부동산 중개·컨설팅** (상가 전문 약 5,000개)
   - 3차: **프랜차이즈 본사 입지팀** (점포 50개+ 본사 약 1,500개)
   - 4차: **지자체 골목상권 정책** (공공 입찰)

5. **수익모델 (구체 수치)**:
   - B2C 1회 리포트: **9,900원** (좌표 1개 + 업종 1개) / 패키지 5건 39,000원
   - B2B 컨설팅 좌석: **월 99,000원** (월 100건 리포트 포함)
   - 프랜차이즈 본사: **연 600~1,200만원** (점포수 따라)
   - 지자체 입찰: 연 3,000~1억원 (정책분석 패키지, B2G — v2)
   - 가정: 1년차 B2C 5,000건 + 컨설턴트 50명 + 본사 3개 = (5000×9900/12) + (50×99000) + (3×600만/12) = MRR 약 1,180만원

6. **차별화 포인트 (moat)**:
   - **누적되는 grid 라벨 데이터**(예측↔실제 결과)가 시간 갈수록 자산화. 1~2년 차에 모델 정확도 격차 결정적
   - 호갱노노·디스코는 거주용·실거래가 중심 → 상업 입지 모델은 분산투자 어려움 (메인 BM과 분리)

7. **기술 난이도 & MVP 개발 기간**: **5~6개월 (MEDIUM)**
   - 1단계 (3개월): 5개 레이어 (사업자·인구·주소·기상·재해) MVP — 그리드 매칭 + 단순 가중평균

   - 3단계 (1개월): 결제 + 리포트 PDF 생성

8. **리스크**:
   - 정확도가 미흡할 경우 1회 결제 후 재구매 없음 → **"점수+해석"의 신뢰 디자인** 핵심
   - SK텔레콤·KT 유동인구 데이터 없이는 정확도 한계 — Pro 플랜에서만 결합
   - 호갱노노·직방이 상업 입지로 확장하면 위협 — 하지만 거주용 brand·UI 분리 어려움이 진입장벽

## 아이디어 ④. **"우리 자녀에게 맞는 공모전·장학금·교육사업 자동 큐레이션" — KidPath**

## 아이디어 ⑤. **"정책 ROI 트래커" — PolicyLens**
### 패턴 D: 정책 효과 측정 (Causal Inference)

1. **One-liner**: **지자체 정책기획·평가팀, 시민단체, 기자, 국회 보좌관**에게 **중앙·지자체 정책 발표 → 예산 집행 → 결과 데이터(사업자·고용·매출·인구·환경 등) 변화를 시계열·공간으로 묶어 "이 정책에 100억 썼더니 실제로 무엇이 변했는가" 시각화 + 자동 리포트**를 제공한다.

2. **사용 데이터셋**:
   - 정책 발표 (정부24·각 부처·지자체 보도자료 — 스크래핑)
   - 열린재정 (`openfiscaldata.go.kr`) — 예산 편성·집행
   - 사업자 등록·폐업 (#4) — 정책 전후 사업자 변동
   - KOSIS 고용·인구·매출·소득 (#10)
   - 국토부 부동산 (#2) — 부동산 정책 효과
   - 환경부 대기질 (#6)·교통사고 통계 — 환경/교통 정책 효과
   - 도로명주소 (#3) — 정책 시행 행정구역 정확 매핑
   - 결합: 한국은행·BC카드 지역·업종 매출 (분기 공시)
   - 결합: 통계청 마이크로데이터 (협력 시 — 가구·기업 단위)

3. **🔑 결합 시너지 메커니즘**:
   - 열린재정 단독 → "100억 편성·집행"만 앎 (input)
   - 사업자 통계 단독 → "강남구 사업자 +5%"만 앎 (outcome but unattributed)
   - **시간(정책 전후) + 공간(시행 지역 vs 비시행 통제집단) + 5~10종 결과 데이터 결합** → "강남구 골목상권 활성화 사업 100억 → 시행 6개월 후 사업자 +12% (통제집단 대비 +7%p), 카드매출 +8%, 평균 임대료 +15% (gentrification 신호)" 형태의 **차분 추정 (Difference-in-Differences) 자동 시각화**
   - 학계에서는 사후 박사논문/KDI 보고서로만 존재 → **실시간·시각화·다중지표 SaaS는 미존재**
   - 단일 데이터로는 인과추정 불가능 — 통제집단 매칭과 다종 결과 결합이 필수

4. **타겟 사용자 & 페르소나**:
   - 1차: **기초·광역 지자체 정책기획·평가팀** (전국 기초 226개 + 광역 17개 = 243개) — 매년 자체평가·국정평가 부담
   - 2차: **시민단체·정책 컨설팅** (참여연대·KDI 출신 컨설팅 등 약 50개)
   - 3차: **국회 보좌관·기자** (Free + 광고/구독 차별화)
   - 4차: **정당 정책연구소** (선거 전 큰 수요)

5. **수익모델 (구체 수치)**:
   - 지자체 라이선스: **연 600~1,200만원** (광역) / **연 300만원** (기초) — 전국 1% 침투 = 2.4개 광역 + 23개 기초 = 연 8,400만원
   - 컨설팅·정당 좌석: **월 199,000원**
   - 기자/보좌관 무료 (viral 채널) + 프리미엄 분석 보고서 건당 50,000원
   - B2B API 판매 (정책연구원·언론사): 호출당 100원 또는 월 정액
   - 가정: 1년차 지자체 10개 + 컨설턴트 30명 = (10×500만/12) + (30×199,000) = MRR 약 1,000만원

6. **차별화 포인트 (moat)**:
   - 한국에 **실시간 정책 ROI 트래커**는 사실상 부재 (KDI·감사원은 사후 보고서)
   - 정책-결과 매칭 데이터셋 자체가 **국가급 자산** — 후발 주자 진입 시 5년+ 격차
   - 기자·시민단체 무료 활용 → 자체 viral 마케팅 (수수료 없는 유기적 노출)
   - 지자체 평가시즌(매년 11~12월) 정기 매출 사이클
   - 미국 USAFacts, 영국 Open Data Institute 등 글로벌 벤치마크는 있으나 한국 특화 부재

7. **기술 난이도 & MVP 개발 기간**: **6개월 (HIGH)**
   - 1단계 (2개월): 5개 데이터 ETL 파이프라인 + 정책 보도자료 LLM 정형화
   - 2단계 (2개월): DiD 통계엔진 (R/Python statsmodels) + 통제집단 자동 매칭
   - 3단계 (2개월): Next.js dashboard + 자동 리포트 PDF 생성
   - 도메인 지식 필요 (계량경제·인과추론) — 1인 개발자 시 자문/공동창업자 1인 권장

8. **리스크**:
   - **정치적 민감성**: 정책효과가 부정적으로 나오면 발주 지자체와 충돌 → "데이터 그대로 보여주고 해석은 사용자 몫" 중립 설계 필수
   - 지자체 조달 사이클(매년 1~3월)에 맞춰 영업해야 — 시즌성
   - 통제집단 매칭의 통계적 타당성 비판 가능 → **방법론 투명 공개 + 학계 자문**으로 신뢰 구축
   - 통계청·KDI가 동일 SaaS 출시 시 위협 — 하지만 공공 UX·민첩성 한계로 1~2년 안전
   - 마이크로데이터 협력이 어려울 시 광역 단위로 한정될 위험

2. **B2C 1회 결제(9,900원) → B2B 컨설턴트 좌석(99,000원/월) → B2G 지자체 입찰(수천만원)** 매출 단가 점진 상승 가능 — 1인 개발 친화적 funnel
3. 누적 grid 라벨 데이터(예측 vs 실제 결과)가 **1~2년차에 후발주자 추격 불가**한 자산이 됨
4. 한국 공공데이터의 가장 큰 강점(주소·위치·통계의 정형성)을 최대 활용하므로 **데이터 정책 변경 위험 가장 낮음**

## v1 → v2 핵심 차이 요약

| 측면 | v1 (단일 데이터 + 알림) | v2 (다중 결합 시너지) |
|---|---|---|
| 결합 데이터 평균 | 1.6개 | 4.4개 |
| 가치 창출 메커니즘 | 모니터링·알림 | 예측·점수화·인과추정·자동 매칭 |
| 단일 베팅 추천 | ① BizGuard (3개월 MVP, 단순) | ③ StreetLens (5~6개월 MVP, 진입장벽 高) |
| Moat 형성 | 운영 노하우 | **데이터 결합 자체 + 누적 라벨** |
| MVP 기간 평균 | 3.8개월 | 5.4개월 |
| 1년차 MRR 추정 평균 | 약 1,200만원 | 약 1,800만원 |

> v2는 v1보다 **MVP 기간 +1.5개월 / 결합 복잡도 +3배**의 비용을 치르는 대신, **단일 데이터로 모방 불가능한 진입장벽**을 처음부터 설계한다.

- [x] **결합 시너지가 진짜 moat인가, 아니면 수사적 결합인가?** — 각 아이디어에 "왜 단일 데이터로 불가능한가" 별도 항목으로 검증. ②~⑤는 결합이 빠지면 가치의 70~90% 손실, ①은 약 60% 손실(가장 약한 결합).
